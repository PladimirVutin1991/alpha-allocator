\# ISL Strategic Wealth Platform
\#\# Research Question
Can a Random Forest classifier using institutional technical indicators (Golden Cross, RSI) outperform the S&P 500 in a core-satellite wealth management portfolio?
\#\# Project Structure
\- \`main\.py\`: Entry point\. Handles installation\, training\, and launching the dashboard\.
\- \`src/\`: Source code for data processing\, modeling\, and simulation\.
\- \`data/\`: Local storage for dataset \(auto\-generated\)\.
\- \`environment\.yml\`: Dependency file for Conda users\.
\#\# How to Run \(Easiest Way\)
1\. Open a terminal\.
2\. Run the following command:
\`\`\`bash python main.py
